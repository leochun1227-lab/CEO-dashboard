# Publish Setup

The public site is read-only. It loads published modules and links from
`portal-data.js`.

## Local Editing

1. Open `index.html` from this repository, or serve the folder locally.
2. Turn on `Editor`.
3. Add or edit modules and links.
4. Click `Export Data` and write the exported file as `portal-data.js`.
5. Commit and push `index.html`, `portal-data.js`, and any dashboard `.html` or `.css` pages.
6. Deploy the site.

On the deployed URL, editor controls are hidden. Visitors only see the content
from `portal-data.js`.
